# Biz Educational Center CRM

O'quv markazlari uchun professional CRM tizimi.

## Texnologiyalar

- **React 19** - UI kutubxonasi
- **TypeScript** - Type-safe dasturlash
- **Vite** - Tez build tool
- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - UI komponentlar kutubxonasi
- **React Router DOM** - Client-side routing
- **Lucide React** - Icon kutubxonasi

## Loyiha Strukturasi

```
src/
├── components/      # Qayta ishlatiladigan UI komponentlar
├── layouts/         # Layout komponentlar
├── pages/          # Sahifa komponentlar
├── routes/         # Routing konfiguratsiya
├── hooks/          # Custom React hooks
├── services/       # API services
├── utils/          # Utility funksiyalar
├── types/          # TypeScript type definitions
├── assets/         # Statik fayllar (rasmlar, fontlar)
├── styles/         # Global CSS fayllar
└── constants/      # Konstantlar va konfiguratsiyalar
```

## O'rnatish

Dependencylarni o'rnating:

```bash
npm install
```

## Ishga tushirish

Development rejimida ishga tushirish:

```bash
npm run dev
```

Build qilish:

```bash
npm run build
```

Preview qilish:

```bash
npm run preview
```
